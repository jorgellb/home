# Accessibility Rules

## WCAG 2.1 AA Compliance

All sites must meet WCAG 2.1 Level AA.

## Required Elements

1. **Skip Link** - First focusable element
2. **Focus Indicators** - Visible on all interactive elements
3. **Alt Text** - All meaningful images
4. **Labels** - All form inputs
5. **Landmarks** - header, main, footer, nav

## Keyboard Navigation

- All functionality via keyboard
- Logical tab order
- No keyboard traps
- Focus visible at all times

## Color Contrast

- Text: 4.5:1 minimum
- Large text: 3:1 minimum
- Interactive elements: 3:1 minimum

## Screen Reader Support

- Semantic HTML5 elements
- ARIA only when needed
- Meaningful link text
- Form error announcements
